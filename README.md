# wp-plugin-kit

**Starter-kit voor Dr.Docks WordPress-plugins**: een schoon vertrekpunt met de gedeelde toolchain al ingericht.

- `src/` — basisklassen: `kit-admin-menu`, `kit-helpers`, `kit-settings`, `kit-updater` (self-update).
- Composer + PHPCS (`phpcs.xml.dist`) + PHPStan (`phpstan.neon`) voorgeconfigureerd.

Kopieer, hernoem en bouw je plugin verder uit. Zie [CLAUDE.md](CLAUDE.md).
